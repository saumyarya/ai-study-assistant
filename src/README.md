# AI Study Assistant

A simple React application that uses the Gemini API to help students learn.

## Features

### Explain Concept
Explains a topic in simple language.

### Summarize Text
Converts long text into short bullet points.

### Generate Quiz Questions
Creates quiz questions from a topic.

## Technologies Used

- React
- CSS
- Gemini API

## How to Run

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the local URL shown in the terminal.

Recursion
